Metrotvnews.com, Monte Carlo: Petenis putra nomor satu dunia asal Serbia Novak Djokovic memastikan gelar Monte Carlo Masters setelah unggul dua set langsung 6-2, 7-6 atas Rafael Nadal di laga pamungkas, Minggu (21/4).
Djokovic tampil mendominasi, khususnya di set pertama. Memasuki set kedua, Nadal memberi perlawanan sengit. Tetapi Djokovic akhirnya memastikan kemenangan melalui adu servis (tie break), untuk menutup set penentuan itu.
